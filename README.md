# Aplikasi CKG Umum Puskesmas Kosambi

Klik tombol di bawah ini untuk memasang script pendaftaran dan skrining otomatis:

[![Instal Script](https://img.shields.io/badge/INSTAL-SCRIPT-green?style=for-the-badge&logo=tampermonkey)](https://raw.githubusercontent.com/adihpkm/ckgumum.app/refs/heads/main/pasang_ckg.user.js)

### Cara Penggunaan:
1. Pastikan sudah ada ekstensi **Tampermonkey**.
2. Klik tombol **INSTAL SCRIPT** di atas.
3. Klik **Install** pada halaman yang muncul.
4. Buka website [ASIK Kemkes](https://sehatindonesiaku.kemkes.go.id/).
